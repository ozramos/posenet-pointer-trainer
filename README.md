# Local Development
## Prereqs
First, install [Node.js](https://nodejs.org/en/download/) >= 8. Then run any of the following scripts from the command line:

```bash
# Start a file watch server
npm run serve

# Compile the app for production
npm run build

# RUn linters
npm run lint
```
